Projects

sPorts - Smart Ports

This project was part of my third semester assignment at UCN where with as a group we were tasked to develop a distributed web as well as desktop application in .NET. We came up with an idea to create a software for marina owners where they can add and manage their marinas and for boat owners where they can add their leisure boats and book specific marinas that are part of our system.

Support Ticket System

This project is part of my second semester assignment at UCN where with as a group of students we were tasked to develop Java applications for a company which in our case was a polish company Grast-MTB and the software is support ticket system for better handling of customer's questions, complaints, etc.

Sky Run (Unity3D game)

Sky Run is another Android game I have published on Google Play Store. The game idea is similar to Comet Run, but now you play as a ball high in the clouds. You need to change positions swiftly and regurarly, because the cubes, you are standing on, fall after a few seconds.

Esp8266 Temperature Project

Simple project that uses temperature sensor with which you can measure temperature in the room or outside, store the data from the sensor in a database and then display a graph with those values on a webpage.

Comet Run (Unity3D game)

Comet Run is the first Android game I have published on Google Play Store. It is a simple survival game made in Unity3D environment. You play as a comet and try to avoid asteroids that fly around you.
